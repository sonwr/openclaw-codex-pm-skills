# PM Prioritization Helper

## Purpose
Rank initiatives with transparent criteria and decision rationale.

## Use when
- Backlog prioritization
- Quarterly planning
- Trade-off discussions

## Workflow
1. List candidate items
2. Score impact, effort, risk, alignment
3. Compute weighted score
4. Flag dependencies/blockers
5. Recommend top set for current cycle

## Output
- Prioritized list with scores
- Excluded items and reasons
- Next review triggers
