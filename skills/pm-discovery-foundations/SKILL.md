# PM Discovery Foundations

## Purpose
Guide idea discovery with explicit assumptions, risk ranking, and experiment plans.

## Use when
- Evaluating a new product idea
- Clarifying risky assumptions
- Planning first validation experiments

## Workflow
1. Clarify user/problem/job-to-be-done
2. List assumptions (value, usability, feasibility, viability)
3. Rank by impact × uncertainty
4. Propose 3 low-cost experiments
5. Define success/failure criteria

## Output format
- Problem statement
- Assumption table
- Top 3 experiment plan
- Decision recommendation (continue/pivot/stop)
