# Governance sandbox minimum win status note

Use this note when the workstream needs the smallest honest progress statement before broader governance-sandbox scope.

Call it a minimum win only when the slice still proves:
- one scenario file import path
- one generated report artifact lane (JSON plus markdown/html when available)
- one replayable validation command or test tied to that same slice

Do not call preset expansion or web-demo work the current minimum win until the scenario-file and report-bundle lane is already green.
