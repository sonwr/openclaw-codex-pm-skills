# Governance Sandbox Scenario Report Owner/Audience Review Loop

Use this note when one imported scenario file already produces a JSON/Markdown/HTML report bundle and you want a reviewer-ready recheck before widening preset or web-demo scope.

Keep the loop small:

1. confirm the scenario source path is still visible,
2. confirm report owner and audience routing still read clearly,
3. confirm the generated JSON/Markdown/HTML bundle paths still match that same replay.

If one of those three cues drifts, repair the scenario-file -> report-bundle lane before expanding aliases, presets, or UI work.
