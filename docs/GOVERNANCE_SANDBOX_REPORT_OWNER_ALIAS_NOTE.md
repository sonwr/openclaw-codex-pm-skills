# Governance Sandbox report owner alias note

Keep governance-sandbox scenario files owner-aware without widening the first report loop.

Use one of `report.owner`, `report.owners`, `report_owner`, or `report_owners` when the markdown/html memo should show who owns the rehearsal handoff.
That owner cue should stay beside the existing report title / audience metadata so reviewers can reopen the artifact bundle without guessing the maintainer lane.
