# Install on OpenClaw (Codex-focused setup)

## 1) Copy skills

Copy folders from `skills/` into your OpenClaw skill location.

## 2) Enable for your agent

Ensure your target agent can access these skills through your OpenClaw configuration/allowlist.

## 3) Validate

- Start a session
- Ask for a PM task (discovery/strategy/PRD)
- Confirm the skill behavior is applied consistently

## 4) Safety checks (recommended)

For shared/group contexts:

- Keep filesystem scope narrow
- Restrict destructive tools
- Avoid exposing sensitive memory sources
