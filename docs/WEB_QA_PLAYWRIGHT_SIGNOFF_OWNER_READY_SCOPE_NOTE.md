# Web QA Playwright signoff owner-ready scope note

Use this note when validator PASS is real, the owner is known, but the handoff still needs to stay lane-scoped rather than sounding like a full replay signoff.

- Keep the lane, owner, and artifact visible in the first sentence.
- Say what is ready now, not what might be ready after a wider rerun.
- If unresolved failures remain outside the lane, say they stay out of scope for this handoff.
