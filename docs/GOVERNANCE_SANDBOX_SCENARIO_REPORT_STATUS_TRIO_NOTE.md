# Governance sandbox scenario report status trio note

Use this when one imported scenario file already generates the JSON / Markdown / HTML report bundle and you need a short PM-facing status line.

Keep the status line anchored to three facts in this order:
- imported scenario source
- shared report basename
- generated JSON / Markdown / HTML artifacts

Example:

`Scenario replay is ready: examples/dao-upgrade.yaml produced the dao-upgrade-review bundle with JSON, Markdown, and HTML artifacts for review.`
