# Governance Sandbox scenario report output review

Use this note when a PM-facing change touches governance-sandbox scenario-file proof.

Minimum review lane:

1. replay one imported scenario file,
2. regenerate the JSON/Markdown/HTML report bundle,
3. confirm the basename and artifact paths still read clearly in the handoff,
4. rerun the validator tests before merging.

This keeps scenario-input evidence and report-output evidence coupled in the smallest honest review loop.
