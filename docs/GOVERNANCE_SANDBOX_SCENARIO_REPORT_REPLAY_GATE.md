# Governance Sandbox Scenario Report Replay Gate

Use this note when you want a compact PM-facing gate for one scenario file replay that still proves the JSON, Markdown, and HTML report bundle stayed attached to the same artifact basename.

Replay check:
1. Run one scenario JSON or YAML fixture.
2. Confirm the JSON artifact was regenerated.
3. Confirm the Markdown artifact was regenerated.
4. Confirm the HTML artifact was regenerated.
5. Keep the scenario file path and report basename visible in the handoff.
