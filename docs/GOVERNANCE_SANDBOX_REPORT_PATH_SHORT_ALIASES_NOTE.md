# governance-sandbox report path short aliases note

Use this when a governance-sandbox scenario fixture should stay compact but still emit a full JSON/Markdown/HTML report bundle.

Short aliases that keep scenario files terse:

- `report.json_output`
- `report.md_file`
- `report.html_output`

Reviewers should still confirm the generated report metadata exposes the resolved artifact paths so the bundle stays replayable.
