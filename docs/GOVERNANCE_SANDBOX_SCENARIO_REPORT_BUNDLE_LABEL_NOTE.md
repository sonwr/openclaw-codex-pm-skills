# Governance Sandbox scenario report bundle label note

Keep one scenario replay tied to one stable report bundle label so PM handoff copy does not drift across JSON, Markdown, and HTML artifacts.

Use the same bundle label when you mention:

- the scenario file that was replayed,
- the JSON result,
- the Markdown memo,
- and the HTML review page.

A short, reusable pattern is: `scenario replay: <label> -> report bundle: <basename>`.
