# Governance Sandbox report metadata card

Use this note when reviewing a governance-sandbox report bundle before handoff.

Checklist:
- Confirm the scenario file path is visible in the report metadata.
- Confirm the report basename matches the intended review title or slug.
- Confirm JSON, Markdown, and HTML artifact paths are all visible when those files were generated.
- If any artifact path is missing, keep the handoff scoped to the files that actually exist.

Why it matters:
A report bundle is easier to replay when reviewers can see exactly which scenario input and artifact outputs belong together.
