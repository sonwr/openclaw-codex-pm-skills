# governance sandbox scenario report reviewers note

Use this note when one imported governance-sandbox scenario already generates a reusable JSON/Markdown/HTML report bundle and you want the reviewer audience to stay explicit in the handoff.

## Keep together

- the imported scenario file,
- the shared report bundle basename,
- and the reviewer audience that should open the memo first.

## Template

`Scenario replay: <scenario file> -> <bundle basename> (<reviewer audience>) keeps JSON/Markdown/HTML artifacts aligned for the same governance handoff.`

## Example

`Scenario replay: examples/scenario-report-bundle.yaml -> delegate-confidence-rehearsal (governance reviewers) keeps JSON/Markdown/HTML artifacts aligned for the same governance handoff.`
