# Governance-sandbox scenario/report minimum progress card

Use this card when governance-sandbox work needs the smallest acceptable proof without drifting into preset expansion or web-demo scope.

Keep the slice honest:

- one imported scenario file,
- one generated report bundle (JSON + Markdown + HTML),
- one validation command that proves the bundle still replays.

If any artifact is missing or validation fails, hold the push and report the blocked lane instead of widening scope.
