# Governance sandbox scenario source alias note

For stdin-driven governance-sandbox replays, keep scenario provenance visible even when the CLI path is absent.

Preferred fallback aliases:

- `scenario_source`
- `source`
- `source_file`
- `scenario_path`
- `source_path`

Use the same label across JSON, markdown, and HTML artifacts so PM handoff copy can point back to the original scenario asset without guessing.
