# Governance Sandbox Web Demo Result Download Check

Keep the first web demo slice honest and reviewable:

1. Load one replayable scenario fixture.
2. Confirm the result card shows the recommendation before any download action.
3. Confirm the download affordance is visible on the same result view.
4. Verify the downloaded artifact maps back to the same scenario/report bundle names.
5. If any step breaks, hold the handoff until the result card and download path are aligned again.
