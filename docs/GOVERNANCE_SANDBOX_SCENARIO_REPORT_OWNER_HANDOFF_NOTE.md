# Governance-sandbox scenario report owner handoff note

Use this note when one governance-sandbox replay already produced a shared JSON, Markdown, and HTML report bundle and you need a short owner-facing handoff.

Keep the handoff scoped to:
- one scenario file,
- one report bundle basename,
- one current owner or reviewer,
- and one next action.

Preferred one-line shape:

> Scenario replay is current, the shared JSON/Markdown/HTML bundle is ready, and the next owner action is explicit.
