# Governance Sandbox scenario-report progress note

For the first delivery lane, keep progress anchored to one real scenario file and one report bundle.

Quick loop:

1. Load one JSON or YAML scenario file.
2. Generate JSON + Markdown + HTML artifacts from the same run.
3. Verify the report paths are stable and reviewable before widening scope.
4. Only then expand into broader UI/demo proof.

This keeps scenario-input work and report-output work coupled in a small, reproducible slice.
